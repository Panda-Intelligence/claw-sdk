# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Pledge

We pledge to act and interact in ways that contribute to an open, welcoming, diverse, inclusive, and healthy community.

## Our Standards

- Be respectful in all interactions.
- Assume positive intent and discuss ideas constructively.
- Accept and give feedback gracefully.
- Focus on what is best for the community.

## Enforcement

If you experience or witness unacceptable behavior, please open a private report via project maintainers in GitHub.
